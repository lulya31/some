= WMS - Система управления складом

{empty} +

****
*WMS*-система управления складом (**W**arehouse **M**anagement **S**ystem) – система, предназначенная для автоматизации управления процессами склада и работы складского комплекса. +
****

== Подключение к Safe Mobile

=== Для Android

****
В браузере телефона перейдите по ссылке:

[source, html]
----
https://mail1.rolf.ru/login
----

Подтвердите, что вы доверяете сертификату и хотите перейти на сайт. Далее необходимо установить на телефон приложение Монитор, загрузка и установка начнется по нажатию на кнопку.
****

****
image::wms_1.png[title="", align=center]
****

****
После установки приложения его необходимо открыть и выбрать пункт `«Доменная учетная запись»`
Заполнить поля:
====
* *URL* сервера:
[source,html]
----
https://mail1.rolf.ru
----
* *Логин*:
[source,html]
----
ваш_логин@int.rolfcorp.ru
----
(Логин чувствителен к регистру, поэтому необходимо вводить его в том виде, в котором он у вас сделан. В компании все логины имеют вид `AAIvanov`, где первые буквы имени, отчества и фамилии большие, если вы получаете ошибку на моменте входа – напишите письмо на группу поддержки: *##UEM#* (`[.red]#uem@rolf.ru#`))

== Общая информация о системе

****
Пользователю с ролью `«Логист»` доступны в системе несколько экранов:
====
* Назначение в ДЦ
* Автомобили
* Загрузка стока
* Уведомления
* Группа экранов `Задачи` с сортировкой всех задач в системе
====
Пользователю с ролью `«Техник»` доступны в системе 2 экрана:
====
* Все задачи
* Мои задачи
====

В верхней части экрана доступны кнопки:
•	«Добавить» - позволяет вручную создать карточку А/М
•	«Обновить» - обновляет текущий список А/М
•	«Загрузить А/М» - импорт Excel-файла со списком заказанных А/М

В нижней части экрана указана вся информация по автомобилю. На вкладке «Информация о заказе» (Рисунок 5) указана информация о заказе А/М (номер, дата заказа, дата резерва и т.д.).

Рисунок 6 (Справочная информация)
На вкладке «Статусы А/М» (Рисунок 7) указана вся информация по изменениям статуса авто и кем были инициированы изменения.
 
Рисунок 7 (Статусы А/М)
На вкладке «Задачи и подзадачи» (Рисунок 8) есть список всех (текущие и завершенные) задач по выбранному А/М. На этой вкладке логист создаёт задачи и подзадачи на техников, о чем будет описано ниже.
Вкладка поделена на 2 части, слева – список всех основных задач. Справа – список подзадач в выделенной, в левой части, основной задачи. По кнопке «Добавить» создаётся задача на техников.

=== Группа экранов Логиста «Задачи»

В этой группе находятся экраны со списками всех задач (текущих и завершенных) по каждому А/М (Рисунок 11).
 
Рисунок 11 (Все задачи)

=== Экран Техника «Все задачи»

Экран «Все задачи» – список всех доступных задач на текущей локации пользователя (Рисунок 12).
 
Рисунок 12 (Все задачи)
 Задачи появляются в списке сразу после их создания в системе другими пользователями или поступают из сторонних систем. В колонках списка указана информация, которая требуется в первую очередь. В нижней части экрана указана вся необходимая информация по автомобилю (Рисунок 13).
 
Рисунок 13 (Подробности А/М)
 В списке могут быть как задачи, так и подзадачи. Подзадачи включают в себя действия, которые необходимо произвести в рамках основной задачи. 
	Например, задача по Предпродажной подготовке включает в себя отправку А/М на сервис (подзадача «Сервис») и последующую мойку (подзадача «Мойка»).
Слева у каждой задачи есть значок  , по которому доступны действия (Рисунок 14):
•	Взять в работу - С помощью этого действия можно взять в работу только одну, основную задачу.
При клике, в работу будет взята только задача «Предпродажная подготовка». Этот вариант нужен, если подзадачи будет выполнять другой техник.
•	Взять всё в работу – С помощью этого действия можно взять в работу основную задачу, а также все связанные с этой задачей подзадачи. 
При клике, в работу будут взяты задача «Предпродажная подготовка» и 2 подзадачи «Сервис» и «Мойка». Если весь процесс по предпродажной подготовке будет выполнять один человек, то удобнее взять всё в работу сразу.
После взятия в работу, задача будет перемещена на экран «Мои задачи».

=== Экран Техника «Мои задачи»

На экране «Мои задачи» - список задач, которые пользователь взял в работу (Рисунок 15).
 
Рисунок 15 (Мои задачи)
В списке отображены автомобили, по которым необходимо выполнить работу. При выборе авто, в нижней части страницы появляется вся необходимая информация по нему (Рисунок 3). 
Чтобы просмотреть список всех задач по автомобилю, необходимо перейти на вкладку «Задачи и подзадачи» (Рисунок 16). 
 
Рисунок 16 (Задачи и подзадачи)
Слева у каждой задачи есть значок  , по которому доступны действия (Рисунок 17):
•	Изменить – для работы с задачей/подзадачей.
•	Отпустить – для прекращения работы над задачей и возврата её в общей список, чтобы задачу/подзадачу мог взять другой техник.
